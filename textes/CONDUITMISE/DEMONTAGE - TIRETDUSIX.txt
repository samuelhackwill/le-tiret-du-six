0/ Caisses de rangement
	apporter les caisses
		ordi
		alimOrdi+acsr
	disposer les caisses environ par commune dans la salle

1/ Ordis :
	vider le cache
	éteindre
	débrancher alim et RJ45
	ranger ordi dans marinade
	glisser marinade(ordi) dans caisse(ordi)
		environ 6 ordis par caisse
		ranger ensemble les ordis d'une commune

2/ BUREAUX - plateaux
	délicatement les désassembler
	les stacker au sol
		face intérieure vers le sol
		triés par taille
		dans coin ne gênant pas circulation

3/ alims des ordis :	
	débrancher les alims des ordis 
	les ranger dans leur ziplok attitrée
	ranger les ziplok(alimOrdi) dans la caisse(alim+acsr)

4/ RJ45 - bloc 1 :
	débrancher les RJ45 
	les trier par taille et par couleur
		bleu : 1,5m, 3m
		gris : 7,5m
	les ranger dans des ziplok
	ranger les ziplok(RJ45) dans marinade de sa taille

5/ Accessoires :
	ranger switch + alim dans ziplok
	ranger ziplok(switch+alim) dans caisse(alim+acsr)

6/ Multiprises :
	débrancher
	rouler brancher sur soi
	ranger les alims dans caisse ?

7/ BUREAUX - gouttières
	désassembler TRÈS délicatement
		synchronisées une personne de chaque côté
	les stacker au sol
		face intérieure vers le sol
		triées par taille
		dans coin ne gênant pas circulation

8/ BUREAUX - pieds
	ouvrir clapet pied
	retirer les clés du dessus
	retourner
	retirer les clés du dessous
	ranger les clés dans boite(clés) et marinade(clés)
	déplier les pieds et les disposer au sol
		face intérieure vers le sol
		dans coin ne gênant pas circulation

9/ RJ45 - bloc 2 :
	débrancher les RJ45 
	les trier par taille
		gris : 7,5m, 15m, 20m
	les ranger dans des ziplok
	ranger les ziplok(RJ45) dans marinade de sa taille
